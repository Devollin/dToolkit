Bind

This item only works when running on the client. Client

A Bind class that is used as an interface for binding keys to ContextActionService.

Types​

Keys​

</>
type Keys = {Enum.KeyCode}

A list of Enum.KeyCode to bind an action to.

BoundFunction​

</>
type BoundFunction = (
actionName: string,
state: Enum.UserInputState,
input: InputObject
) → ()

A function used to bind context actions to.

Properties​

boundFunction​

</>

Function bound to the given Keys.

keys​

</>
Bind.keys: Keys

Keys bound to the action.

Functions​

new​

</>
Bind.new(
name: string,--

The name of the action.

keys: Keys,--

The list of keys you want to bind.

boundFunction: BoundFunction?,--

Function that the binding is bound to.

enabled: boolean?,--

Determines if the binding should be enabled upon creation; defaults to true.

mobileButton: boolean?--

Determines if a mobile button should be added with this binding; defaults to false.

) → Bind

Returns a new Bind class.

local newBind = Bind.new("Test", {Enum.KeyCode.E}, function(actionName, state, input)
	print("Wow!")
end)
