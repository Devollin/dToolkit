Timer

A timer class.

Types​

Finished​

</>
type Finished = Signal<>

Fired when the Timer finishes running.

Paused​

</>
type Paused = Signal<>

Fired when the Timer has been paused.

Updated​

</>
type Updated = Signal<number,number>

Fired when the Timer has been updated. The first number is the time remaining, the second is the time elapsed.

Functions​

new​

</>
Timer.new(
duration: number?--

The duration you want to give to the Timer; or nil, if you need to set it dynamically.

) → Timer

Creates and returns a new Timer object.

local newTimer = Timer.new()
newTimer:SetDuration(5)

newTimer.Finished:Connect(function()
	print("Finished!")
	
	newTimer:Start()
end)

newTimer:Start()

GetElapsed​

</>
Timer.GetElapsed(self: Timer) → number

Gets the elapsed time of the Timer. Returns 0 if the duration has not been set or if the Timer finished; otherwise returns the duration.
