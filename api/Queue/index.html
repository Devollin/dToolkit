Queue

A Queue class is used to hold a list of items which are accessible through a first-in, first-out format.

local queue = Queue.new(1, 2, 3, 4)

queue:Add(5)

queue:Next()

print(queue:Get())

queue:Remove(1)

if not queue:IsEmpty() then
	queue:Clear()
end

queue:Destroy()

Get​

</>
Queue.Get(self: Queue<a>) → a

Returns the item at the front of the Queue. This will return nil if the Queue is empty. It is recommended to check if the Queue is empty by using Queue:IsEmpty() before using this. 

print(queue:Get())

Pop​

</>
Queue.Pop(self: Queue<a>) → a

Returns the item at the front of the Queue and removes it from the Queue. This will return nil if the Queue is empty. It is recommended to check if the Queue is empty by using Queue:IsEmpty() before using this. 

print(queue:Pop())
