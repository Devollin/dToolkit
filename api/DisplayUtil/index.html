DisplayUtil

A library of helpful display-related functions.

Functions​

GetTime​

</>
DisplayUtil:GetTime(seconds: number) → string

Returns a string formatted as: xx:xx

local timeString = DisplayUtil:GetTime(630)

print(timeString) --> 10:30

GetTimeWithHours​

</>
DisplayUtil:GetTimeWithHours(seconds: number) → string

Returns a string formatted as: xx:xx:xx

print(timeString) --> 10:30:00

GetNumberSuffix​

</>
DisplayUtil:GetNumberSuffix(
number: number,
places: number?--

The decimal place value to show up to. If not given, will show decimals up to 3. Integers only.

) → string

Returns a shortened form of a number. Ex: 1000 -> 1k, 555555 -> 555k 

local shortenedNumber = DisplayUtil:GetNumberSuffix(5254, 2)

print(shortenedNumber) --> 5.25k

PlaceCommasInNumber​

</>
DisplayUtil:PlaceCommasInNumber(number: number) → string

Places commas within a number, and returns the string. Ex: 1000 -> 1,000, 555555 -> 555,555 

local lengthenedNumber = DisplayUtil:PlaceCommasInNumber(5254)

print(lengthenedNumber) --> 5,254

NumberToRomanNumerals​

</>
DisplayUtil:NumberToRomanNumerals(number: number) → string

Returns a string of roman numerals based off of the number given to it.

local romanNumerals = DisplayUtil:NumberToRomanNumerals(10)

print(romanNumerals) --> X
