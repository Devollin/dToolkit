SimpleValue

An object used to store a single state, and trigger every Connection listening to its Signal members.

local state = SimpleValue.new(false)

state.Changed:Connect(function(newValue, oldValue)
	print(newValue, oldValue)
end)

state:Set(true)

task.wait(1)

local copiedState = state:Clone()

print("This is the current state:", copiedState:Get())

copiedState:Destroy()
String Literals

It should be noted that for one reason or another, SimpleValue is not able to typecheck string literals properly.

There is an alternative type named InternalSimpleValue, which can be used for classes built using it! The primary difference is that InternalSimpleValue drops the :Set(), :Destroy(), and :RawSet() methods, but only from the type, not the object.

Properties​

OnlyUpdateIfDifferent​

Event
</>
SimpleValue.OnlyUpdateIfDifferent: boolean

Enabling this makes it so future attempts to use :Set() or :RawSet() are first compared to the current value; if they are determined to be the same value, then it will not fire the .Changed Signal. 

state.OnlyUpdateIfDifferent = true

Changed​

Event
</>
SimpleValue.Changed: Signal<a,a>

Is fired after the value changes for SimpleValue. The first passed item is the new value, the second is the old value.

state.Changed:Connect(function(newValue, oldValue)
	print(newValue, oldValue)
end)
